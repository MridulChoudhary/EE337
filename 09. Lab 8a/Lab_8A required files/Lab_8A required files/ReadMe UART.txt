For the UART driver installation part for Pt-51
1. Connect the board
2. Install drivers from this file P3200.exe(file is available on teams)
3. Open device manager
4. Reboot the board
   4.1 change the switch to detach
   4.2 press boot
   4.3 press reset
   4.4 release reset
   4.5 release boot
   4.6 change switch to attach
5. Now right click on the unknown device/ unknown drivers in the device manager
6. Click on update drivers
7. Click on browse my computer for drivers  
8. Let me pick ... (or) provide the path where you installed the above P3200.exe application manually and click next.
9. It will show under the model section, version of 2007
10. Select it and install
